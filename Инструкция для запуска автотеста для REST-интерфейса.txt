1) Установите инструмент Postman по данной ссылке - https://www.getpostman.com/downloads/
2) Скачайте из репозитория GitHub файл в формате .json, который содержит коллекцию с одним автотестом (Автотест проверяет создание нового героя в системе)
3) Импортируйте коллекцию в Postman 
4) Откройте её, выберите скрипт "Create a hero  (Autotest)" и перейдите на вкладку "Tests"
5) Нажмите кнопку "Send" 
6) Автотест должен отработать и выдать положительные результаты во вкладке "Test Results"
